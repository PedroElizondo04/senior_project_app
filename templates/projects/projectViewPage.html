ByteSync - A Simple File Synchronization Tool

ByteSync is a command-line tool that allows users to sync files between two directories efficiently. The tool will detect changes in files by using a hashing approach (e.g., SHA-256) to detect file modifications and minimize unnecessary file transfers. Users can configure exclusions via a .bytesync file or .gitignore to prevent certain files from being synchronized. An optional logging system will keep track of synchronization actions.

Skills needed: Rust, Algorithms, Linux, C, Concurrency Programming
