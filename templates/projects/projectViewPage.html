ByteSync - A Simple File Synchronization Tool

ByteSync is a command-line tool that allows users to sync files between two directories efficiently. The tool will detect changes in files (new, modified, or deleted) and update the target directory accordingly. It will support different sync modes, such as one-way sync (source to destination) and two-way sync (bidirectional synchronization).

To improve performance, ByteSync will use hashing (e.g., SHA-256) to detect file changes and minimize unnecessary file transfers. Users can configure exclusions via a .bytesyncignore file, similar to .gitignore, to prevent certain files from being synchronized.

Additionally, an optional logging system will keep track of synchronization actions.

  • Algorithms
  • Linux
  • CLI
  • Concurrent Programming
